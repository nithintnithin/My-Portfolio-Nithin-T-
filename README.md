# ✨ Nithin T | Personal Portfolio Website

A premium, modern, and highly interactive single-page portfolio website showcasing academic history, technical skills, core competencies, certifications, and software projects. 

Designed with modern UI/UX aesthetics featuring a customized dark theme, interactive particle canvas background, a terminal-style mock developer window, smooth page animations, and an interactive certifications lightbox modal.

## 🌟 Key Features

1.  **Dynamic Visuals & Layout:**
    *   **Interactive Particle Canvas:** A custom HTML5 canvas particle network animating behind sections in real-time.
    *   **Premium Glow Effects:** Glowing background spheres utilizing high-end blur styling for visual depth.
    *   **Glassmorphism & Micro-animations:** Modern CSS card structures, button hover animations, and scroll-indicator prompts.
2.  **Interactive Components:**
    *   **Developer Terminal Window:** A code-themed visual element presenting developer parameters (`developer.js`) with custom syntax highlighting.
    *   **Dark/Light Theme Toggle:** A custom dual-state theme switcher adjusting page colors dynamically with persistent transitions.
    *   **Animated Timeline:** A vertical timeline tracing educational checkpoints with custom status chips.
    *   **Animated Skill Bars:** Visual progression indicators for programming languages, database systems, web tech, and core subjects.
    *   **Certificate Lightbox Modal:** An interactive image-viewer allowing users to preview credentials and diplomas in full resolution.
3.  **Functional Utilities:**
    *   **Interactive Resume Download:** Smooth resume download action triggers.
    *   **Validated Contact Form:** Fully responsive, client-side checked text inputs with custom inline errors, submission loading spinner, and success state indicators.
    *   **Scroll Progress Indicator:** A custom top-pinned bar indicating read progress down the page.

## 💻 Running Locally

To run the portfolio website locally on your computer, you can choose any of the following methods:

### Method 1: Double-Click Index File
1.  Clone or download this repository.
2.  Navigate to the repository folder.
3.  Double-click `index.html` to open it in your default web browser.

### Method 2: Live Server (Recommended for Developers)
If you are using **Visual Studio Code**:
1.  Open the workspace folder in VS Code.
2.  Install the **Live Server** extension by Ritwick Dey.
3.  Click the **Go Live** button in the bottom status bar, or right-click `index.html` and select **Open with Live Server**.

### Method 3: Python Local Server
Using command-line interfaces:
```bash
# Python 3
python -m http.server 8000
```
Open your browser and navigate to `http://localhost:8000`.
